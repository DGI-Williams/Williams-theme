Williams theme
==============

D6 Fusion sub-theme for Williams.
(fusion_starter_lite)

Instructions:
-------------

- download and enable fusion 6.x-1.13 or later
- navigate to the themes folder and do
- git clone https://github.com/DGI-Williams/Williams-theme.git
- mv williams-theme williams
- enable the theme
- the main navigation menu is populated with the primary links
- the bottom right corner of the footer will be populated with the secondary links

In the Williams theme settings set where X = checked and O = unchecked
X Logo
O Site name
O Site slogan
O Mission statement
O User pictures in posts
O User pictures in comments
O Search box
X Shortcut icon
X Primary links
X Secondary links 

Fusion settings:
Use default (set to fluid)
- Fusion theme settings > General settings > Layout

Custom Williams theme settings:
- none
